How It Works

  1. Episode Directory - System finds your episode folder (e.g., Matthew Spaur)
  2. Transcript Analysis - Reads the prep conversation transcript to understand context
  3. AI Generation - Claude generates relevant questions based on actual conversation content
  4. Auto-Save - Questions are automatically saved to interview questions.md in the episode folder

Output Format

Generated questions include:

  • Introduction Script - A podcast-ready intro you can read on air
  • Opening Questions (2-3) - Build rapport and ease into conversation
  • Main Discussion (6-8) - Deep dive into guest's expertise and experience
  • Closing Questions (2-3) - Reflections, insights, and forward-looking questions

Requirements

Your episode directory must contain a prep conversation transcript file. Common names:

  • prep conversation transcript.txt
  • prep-transcript.txt
  • Any file matching *pre*.txt or *prep*.txt
